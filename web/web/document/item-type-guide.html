Question Type Instructions Guide

Learn how to create different types of assessment items

Multiple Choice Questions (MCQ)

Multiple Choice Questions (MCQs) present a question with several answer options where only one is correct.

1
Select MCQ from the dropdown menu

In the exam creation interface, find the "Add Question" button and select "MCQ (Multiple Choice Question)" from the dropdown menu.

2
Enter your question

Type your question in the editor field. You can use formatting options to include bold text, italics, or even mathematical equations.

3
Add answer options

By default, four options (A, B, C, D) are provided. You can add more options by clicking the "+" button. To mark the correct answer, select the radio button next to the correct option.

4
Set marks and settings

Assign marks to the question and configure additional settings:

  • Shuffle Question: Randomize the order of this question among others
  • Shuffle Options: Randomize the order of answer options
  • Evaluate Answer: Automatically score the response
Tips for creating effective MCQs
  • Make sure there is only one clearly correct answer
  • Keep all options approximately the same length
  • Avoid using "All of the above" or "None of the above" options
  • Write plausible incorrect options (distractors)
True/False Questions (TF)

True/False questions present a statement that the student must identify as either true or false.

1
Select True/False from the dropdown menu

In the exam creation interface, find the "Add Question" button and select "TF (True/False Question)" from the dropdown menu.

2
Enter your statement

Type your statement in the editor field. The statement should be definitively true or false, without ambiguity.

3
Select the correct answer

Below the statement, select either "True" or "False" as the correct answer.

4
Set marks and settings

Assign marks to the question and configure additional settings as needed.

Tips for creating effective True/False questions
  • Avoid double negatives in your statements
  • Avoid using absolute terms like "always," "never," "all," or "none"
  • Keep statements clear, concise, and unambiguous
  • Make sure the statement is definitively true or false
Fill in the Blanks (FTB)

Fill in the Blanks questions require students to complete sentences by filling in missing words or phrases.

1
Select Fill in the Blanks from the dropdown menu

In the exam creation interface, find the "Add Question" button and select "FTB (Fill in the Blanks)" from the dropdown menu.

2
Enter your question text with blanks

Type your question text, using [blank] wherever you want a blank to appear. For example: "The capital of France is [blank]."

3
Set correct answers for each blank

For each blank you created, you'll need to specify the correct answer(s). You can add multiple correct answers for each blank to account for variations. You can also choose between text input or dropdown options for each blank.

4
Set marks and settings

Assign marks to the question and configure additional settings as needed.

Tips for creating effective Fill in the Blanks questions
  • Keep blanks focused on key concepts or terms
  • Avoid having too many blanks in a single sentence
  • Provide enough context so that the answer is clear
  • Consider alternative correct answers (e.g., "Paris" and "paris" for the capital of France)
Match Following Questions (MTF)

Match Following questions require students to match items from one column with corresponding items in another column.

1
Select Match Following from the dropdown menu

In the exam creation interface, find the "Add Question" button and select "MTF (Match Following)" from the dropdown menu.

2
Enter your question title and instructions

Type a clear title and instructions for the matching exercise. Explain what students need to match and any special rules.

3
Add match pairs

Each match pair consists of a question item and its corresponding answer. Enter the question item in the left field and the matching answer in the right field for each pair. To add more match pairs, click the "+" icon in the attachment container.

4
Configure settings

Assign marks to the question and configure additional settings:

  • Shuffle Question: Randomize the order of this question among others
  • Evaluate Answer: Automatically score the response
Tips for creating effective Match Following questions
  • Keep the number of items manageable (4-8 pairs is ideal)
  • Use clear and concise language for both items and matches
  • Ensure all matches have a single, clear correct answer
  • Use similar length and grammatical structure within each column
Ordering Questions (OR)

Ordering questions require students to arrange a set of items in the correct sequence or order.

1
Select Ordering from the dropdown menu

In the exam creation interface, find the "Add Question" button and select "OR (Ordering)" from the dropdown menu.

2
Enter your question instructions

Type clear instructions explaining what students need to arrange and the basis for the correct order (chronological, size, importance, etc.).

3
Add items to be ordered

Use the input field to add items one by one. Type an item and press Enter or click the plus icon. Each item will appear in the list below where you can arrange them in the correct order.

4
Arrange items in the correct order

Drag and drop the items using the grip lines on the left of each item. The sequence you create will be considered the correct answer. Students will see these items in a randomized order.

5
Configure settings

Assign marks to the question and configure additional settings:

  • Shuffle Question: Randomize the order of this question among others
  • Evaluate Answer: Automatically score the response
Tips for creating effective Ordering questions
  • Use 4-7 items for optimal difficulty (fewer is too easy, more becomes unwieldy)
  • Ensure there is only one logical correct order
  • Use clear, distinct items that aren't ambiguous in their sequence
  • Consider using numbered steps, chronological events, or process sequences
Short Answer Questions (SAQ)

Short Answer questions require students to provide brief written responses, typically a few words or sentences.

1
Select Short Answer from the dropdown menu

In the exam creation interface, find the "Add Question" button and select "SAQ (Short Answer Question)" from the dropdown menu.

2
Enter your question

Type your question in the editor field. Be specific about what you're asking and indicate the expected length or format of the answer. For example: "Define photosynthesis in 2-3 sentences." or "List three causes of World War I."

3
Add attachments (optional)

If your question requires reference materials, click the folder icon to add attachments.

4
Configure settings

Assign marks to the question and configure additional settings:

  • Shuffle Question: Randomize the order of this question among others
  • Evaluate Answer: Enable or disable automatic evaluation (note that short answers typically require manual grading)
Tips for creating effective Short Answer questions
  • Be clear about the expected length of the answer
  • Use precise language that targets the specific knowledge you want to assess
  • Consider including a word limit in the question (e.g., "Answer in 50 words or less")
  • For easier evaluation, phrase questions that have specific factual answers
  • Consider whether automatic grading is appropriate, or if manual review is needed
Programming Questions (PRQ)

Programming questions require students to write code to solve a specific programming problem or implement an algorithm.

1
Select Programming from the dropdown menu

In the exam creation interface, find the "Add Question" button and select "PRQ (Programming Question)" from the dropdown menu.

2
Enter your programming problem

Type your programming problem in the editor field. Include:

  • Clear problem description
  • Input and output specifications
  • Any constraints or requirements
  • Example inputs and expected outputs
3
Select programming language

Use the language selector dropdown to specify which programming language students should use (C, Python, Java, etc.).

4
Test your problem (optional)

Click the "Open Editor" button to test the code execution environment. This helps ensure your problem is correctly configured.

Tips for creating effective Programming questions
  • Provide clear and unambiguous problem statements
  • Include sample inputs and expected outputs
  • Specify any constraints (time limits, memory limits, etc.)
  • Consider providing starter code for complex problems
  • Test your question with various inputs to ensure it's solvable
Table Based Questions (TAB)

Table Based questions require students to fill in or interact with a table structure, useful for financial calculations, data analysis, or structured responses.

1
Select Table Based from the dropdown menu

In the exam creation interface, find the "Add Question" button and select "TAB (Table Based)" from the dropdown menu.

2
Enter your question instructions

Type clear instructions explaining what students need to complete in the table and any specific requirements or context.

3
Insert a table

Click the "Insert Table" button to create a table structure. You can set the number of rows and columns as needed.

4
Configure answer fields

Click "Add Answer Field" to create fields where students will enter answers. For each answer field, specify:

  • Cell Reference: The cell location (e.g., A1, B2)
  • Answer Type: Text, Number, Formula, or Dropdown
  • Correct Answer(s): The expected answer(s)
Tips for creating effective Table Based questions
  • Keep table structures clear and well-organized
  • Provide clear labels for rows and columns
  • Indicate which cells students need to fill in
  • For complex tables, consider providing starter values in some cells
  • Test your table to ensure it displays correctly on different devices
Image Response Questions (IR)

Image Response questions allow students to take and upload a photo as their answer, ideal for documenting experiments, field work, or physical creations.

1
Select Image Response from the dropdown menu

In the exam creation interface, find the "Add Question" button and select "IR (Image Response)" from the dropdown menu.

2
Enter your question instructions

Type clear instructions explaining what students need to photograph and upload. Be specific about what should be visible in the image.

3
Add attachments (optional)

If your question requires reference materials or examples, click the folder icon to add attachments.

Tips for creating effective Image Response questions
  • Be specific about what must be visible in the image
  • Suggest optimal lighting conditions and angles if relevant
  • Provide examples of acceptable images if possible
  • Specify any image quality requirements (resolution, clarity, etc.)
  • Consider whether students need access to a device with a camera
Upload Document Questions (UD)

Upload Document questions allow students to upload files such as ID proofs, certificates, or completed documents as their response.

1
Select Upload Document from the dropdown menu

In the exam creation interface, find the "Add Question" button and select "UD (Upload Document)" from the dropdown menu.

2
Enter your question instructions

Type clear instructions explaining what document students need to upload. Be specific about the type of document required and any format specifications.

3
Add attachments (optional)

If your question requires reference materials or templates for students to download, complete, and re-upload, click the folder icon to add attachments.

Tips for creating effective Upload Document questions
  • Clearly specify the acceptable file formats (PDF, DOCX, JPG, etc.)
  • Mention any file size limitations
  • Provide a template if students need to fill out a specific form
  • Include examples of acceptable documents if appropriate
  • Specify any document quality requirements (resolution, legibility, etc.)
  • Consider privacy concerns when requesting personal documents
