Complete Student Exam Journey

1. Email Verification & Registration

Verify email address, complete profile with National ID/Passport, upload photo with face validation

2. Login to Student Portal

Access portal using college code + email + ID number combination

3. System Compatibility Check

Test webcam, screen sharing permissions, browser compatibility (Chrome/Edge/Brave only)

4. Wait for Exam Finalization

CRITICAL: You CANNOT start exam until faculty finalizes it. Check exam status in your dashboard.

5. Accept Exam Rules & Start

Read and accept 7 mandatory rules, allow full-screen mode, begin exam with live proctoring

6. Complete Exam

Answer questions, use built-in calculator/code editor if needed, auto-save ensures no data loss

7. Submit & View Results

Submit answers, recordings uploaded automatically, view results when published

Exam Lifecycle & States

Critical Exam Rules

The Golden Rule

Students CANNOT start exam until faculty clicks "FINALIZE"
Exam States Explained
  • Draft State

    Exam being created, not visible to students, fully editable

  • Scheduled State

    Exam configured, visible in student list, but cannot be started

  • Finalized State

    Faculty approved, students can now access and start exam

  • Active State

    Exam in progress, faculty can start/stop anytime

  • Completed State

    Exam ended, evaluation phase, no new attempts

Proctoring & Security Rules
